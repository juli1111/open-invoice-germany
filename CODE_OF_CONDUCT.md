# Verhaltenskodex / Code of Conduct

Dieses Projekt folgt dem [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

Wir verpflichten uns, die Teilnahme an unserem Projekt für alle zu einer belästigungsfreien Erfahrung zu machen — unabhängig von Alter, Körpergröße, Behinderung, ethnischer Zugehörigkeit, Geschlechtsidentität und -ausdruck, Erfahrungsstand, Bildung, sozioökonomischem Status, Nationalität, Aussehen, Religion oder sexueller Identität und Orientierung.

**Erwünscht:** Respekt, konstruktives Feedback, Empathie, Verantwortung für Fehler.
**Nicht toleriert:** Belästigung, beleidigende Kommentare, persönliche oder politische Angriffe, Veröffentlichung privater Informationen Dritter.

Verstöße können an die Maintainer gemeldet werden (siehe [SECURITY.md](SECURITY.md) für den Kontakt). Alle Beschwerden werden vertraulich geprüft.

Der vollständige Text: https://www.contributor-covenant.org/de/version/2/1/code_of_conduct/
